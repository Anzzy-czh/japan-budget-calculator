# 日本预算计算器

一个用于计算赴日留学前期预算的小型网页工具。

## 项目简介

这个项目可以帮助用户估算赴日留学的前期费用，并自动换算成日元，判断当前预算是否安全。

目前支持的费用项包括：

- 机票
- 学费
- 首月房租
- 签证费

用户还可以自定义预算上限（日元），页面会根据输入结果自动判断是“预算安全”还是“预算超支”。

## 已实现功能

- 多项费用输入
- 人民币总额自动计算
- 自动换算为日元
- 预算安全 / 超支状态提示
- 自定义预算上限（日元）
- 清空输入内容
- 多文件结构拆分（HTML / CSS / JavaScript）
